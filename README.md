# Telega-Detector-NO-TELEMETRY

**Telega Detector** — это плагин для Telegram-клиентов (AyuGram, exteraGram и аналогичных), предназначенный для обнаружения пользователей, которые используют приложение «Телега».
Эта версия плагина является **полностью очищенной от сторонней телеметрии и сбора статистики**.
### ⚠️ Почему эта версия?
Оригинальный плагин «Telega Detector» содержал скрытый модуль mkStats, который собирал уникальные отпечатки вашего устройства (device_fingerprint), логировал все ваши действия (кого и когда вы проверяли) и отправлял эти данные на сторонний сервер каждые 1.5 секунды.
**Что было удалено в данной версии:**
 * Полностью вырезан модуль MkStatsCoreClient и все его вызовы.
 * Удален сбор и отправка device_fingerprint и user_hash.
 * Удалены фоновые потоки (threading), «стучащие» на сервер сбора статистики.
 * **Результат:** Плагин выполняет свою функцию (проверку), не передавая разработчикам информацию о вашей активности и ваших контактах.
### Как это работает
Плагин использует публичное API службы звонков Одноклассников (calls.okcdn.ru). Он проверяет, зарегистрирован ли ID пользователя в инфраструктуре, которую использует «Телега» для авторизации и звонков. Если ID присутствует в базе данных звонков этой инфраструктуры, плагин помечает пользователя как «использующего Telega».
### Функционал
 * **Детекция:** Автоматическая проверка профилей пользователей.
 * **Визуализация:** Изменение цвета статуса в профиле (красный — использует, зеленый — не использует).
 * **Информирование:** Возможность отправить пользователю автоматическое предупреждение о рисках безопасности, связанных с использованием «Телеги».
 * **Настройка:** Возможность изменить текст предупреждения в настройках.
 * **Безопасность:** Отсутствие трекеров и аналитики.
### Установка
 1. Убедитесь, что ваш Telegram-клиент поддерживает плагины (например, AyuGram или аналоги).
 2. Скачайте файл .py из раздела Releases.
 3. Переместите файл в папку plugins вашего клиента.
 4. Перезапустите клиент или используйте встроенную функцию загрузки плагинов.
### Настройки
 * В меню плагинов откройте настройки **Telega Detector**.
 * Вы можете сбросить кэш пользователей, если данные кажутся устаревшими.
 * Вы можете настроить текст сообщения, которое отправляется при нажатии кнопки «Предупредить».
